# Arduinoのメモ帳

基本的に [Arduino Makefile](https://github.com/sudar/Arduino-Makefile) を使ったArduinoの利用メモです。AVRについては[AVRのメモ帳](https://github.com/mamemomonga/notebook-avr)もご覧下さい。

# Debianへのセットアップ

インストール

	$ sudo apt install arduino-mk

ビルド

	$ make

アップロード

	$ make upload
